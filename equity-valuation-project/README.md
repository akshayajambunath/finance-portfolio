# Equity Valuation Project

## Overview
This project focuses on analyzing and valuing a publicly traded company using financial statement analysis, ratio analysis, and valuation techniques.

The purpose of this project is to simulate the work performed by financial analysts and develop practical financial modeling skills.

---

## Objectives

- Analyze company financial performance
- Calculate key financial ratios
- Evaluate profitability and growth
- Estimate intrinsic value
- Make an investment recommendation

---

## Tools Used

- Microsoft Excel
- Financial statements
- Ratio analysis
- Valuation methods
